POINTERS (ADVANCED)
=============================

POINTER KYA HAI?
Pointer variable address store karta hai.

Example:
int x = 10;
int *p = &x;

POINTER ARITHMETIC

p + 1 → next memory location

NULL POINTER

Pointer jo kisi valid address ko point nahi karta

DANGLING POINTER

Freed memory ko point kare

VOID POINTER

Generic pointer
Kisi bhi type ka address store kar sakta hai

DYNAMIC MEMORY

malloc()
calloc()
realloc()
free()

Memory heap se allocate hoti hai
