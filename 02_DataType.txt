WHAT IS DATA TYPE?
Data type batata hai:
--------------------------
Kaunsa type ka value store hoga

Kitni memory lagegi

Kya range hoga

Kaunse operations allowed hain

BASIC DATA TYPES
-------------------------
char
-----
1 byte

Range: -128 to 127

Single character store karta hai

int
---------
Usually 4 bytes

Whole numbers store karta hai

float
--------
4 bytes

Decimal values store karta hai

Approx 6 digit precision

double
-------------
8 bytes

High precision decimal

Approx 15 digit precision

---------------

unsigned

-------------------

----------------------------

TYPE CONVERSION
------------------
Implicit conversion
Compiler automatically convert karta hai

Explicit conversion
Manual casting
Example:
float x = (float)10/3;

OVERFLOW
--------------
Jab value range ke bahar chali jaye

Example:
unsigned char x = 255;
x = x + 1;
